# 05 - Arquivos

## Definição
Armazenamento a frio (cold storage) para itens inativos, projetos concluídos e interesses abandonados das outras categorias.

## Propósito
Esta pasta reduz ruído visual sem descartar histórico útil. Ela preserva capital intelectual e contexto passado, mantendo o ambiente principal limpo e focado no que está vivo agora.

## Regras de Formatação
- Nenhuma formatação ativa é necessária.
- O objetivo principal é retirar do campo de visão o que não demanda atenção no presente.
- Preserve a estrutura suficiente para permitir recuperação futura.

## O que vai para cá
- projetos concluídos
- projetos cancelados
- áreas encerradas
- referências que perderam relevância
- materiais que você não quer apagar, mas também não quer manter ativos

## Critérios de Movimentação
- Se um projeto for ressuscitado, retire-o dos Arquivos e devolva-o para `01 - Projetos`.
- Se um interesse antigo voltar a ser útil, mova-o para `04 - Referências`.
- Se uma responsabilidade contínua retornar, restaure-a para `02 - Áreas`.

## Princípio operacional
Arquivar não é descartar. É preservar sem competir por atenção.
