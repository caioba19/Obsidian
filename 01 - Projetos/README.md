# 01 - Projetos

## Definição
Esforços de curto prazo com um objetivo específico, um conjunto de tarefas e um prazo de conclusão claro. Este é o ambiente mais acionável do cofre.

## Propósito
Esta pasta concentra trabalho em andamento. Tudo aqui deve estar conectado a entrega, execução, decisão e progresso observável.

## Regras de Formatação
- Trabalhe com Pacotes Intermediários, como esboços, notas de pesquisa e rascunhos.
- Mantenha foco total na execução.
- Organize cada projeto em torno de um resultado claro.
- Use links bidirecionais para invocar notas do `03 - Zettelkasten` sem deslocá-las do seu contexto permanente.
- Reúna aqui apenas o material necessário para avançar o projeto.

## Estrutura sugerida para cada projeto
- objetivo
- próximos passos
- materiais de apoio
- decisões tomadas
- links para notas conceituais relevantes

## Relação com o Zettelkasten
Projetos consomem conhecimento, mas não devem sequestrá-lo. Quando uma ideia for ampla, atemporal e reutilizável, mantenha-a no `03 - Zettelkasten` e apenas faça referência por link.

## Critérios de Movimentação
- Quando o objetivo for atingido, mova a pasta inteira do projeto para `05 - Arquivos`.
- Quando o projeto for cancelado, mova a pasta inteira para `05 - Arquivos`.
- Se um material deixar de ser específico do projeto e se tornar conhecimento geral, extraia a ideia e consolide-a no `03 - Zettelkasten`.

## Princípio operacional
Projetos são temporários, orientados a resultado e devem manter clareza sobre o que precisa ser concluído.
