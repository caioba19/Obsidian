# 06 - Templates

## Definição
Diretório para armazenar esquemas e modelos de documentos reutilizáveis.

## Propósito
Esta pasta reduz fricção e aumenta consistência. Templates aceleram a criação de notas e garantem repetibilidade no layout, na captura de contexto e no preenchimento de metadados.

## Regras de Formatação
- Crie modelos padronizados para os tipos de nota que você usa com frequência.
- Estruture os templates para facilitar preenchimento rápido.
- Mantenha consistência entre título, contexto, links e campos recorrentes.
- Revise os templates quando perceber atrito ou excesso de campos desnecessários.

## Exemplos de templates úteis
- Template de Nota Atômica
- Template de Reunião
- Template de Livro
- Template de Projeto
- Template de Revisão Semanal

## Critérios de Uso
- Use templates quando quiser padronizar criação de novas notas.
- Ajuste cada modelo à sua prática real, evitando burocracia desnecessária.
- Se um template deixar de refletir seu fluxo, simplifique-o ou substitua-o.

## Princípio operacional
Templates devem servir ao pensamento, não engessá-lo. O melhor modelo é o que reduz atrito sem sacrificar clareza.
