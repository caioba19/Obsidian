# 00 - Inbox

## Definição  oi 
O ponto de entrada universal para a captura de ideias, links e notas rápidas (Fleeting Notes).

## Propósito
Esta pasta existe para reduzir a fricção de captura a praticamente zero. Tudo o que surgir no fluxo do dia pode entrar aqui primeiro, sem exigir decisão imediata sobre categoria, estrutura ou destino final.

## Regras de Formatação
- Priorize velocidade acima de organização.
- Registre a ideia crua assim que ela aparecer.
- Não se preocupe com categorização no momento da captura.
- Não exija tags, links ou metadados completos para salvar uma nota.
- Aceite anotações imperfeitas, incompletas e provisórias.

## O que pode entrar aqui
- Ideias soltas
- Links para revisar depois
- Perguntas
- Insights rápidos
- Trechos de texto
- Lembretes
- Rascunhos iniciais

## Critérios de Movimentação
Processe esta pasta regularmente, por exemplo durante a Revisão Semanal.

Ao processar:
- mova para `01 - Projetos` o que estiver ligado a um objetivo com prazo;
- mova para `02 - Áreas` o que representar responsabilidade contínua;
- transforme em notas permanentes no `03 - Zettelkasten` o que merecer síntese e conexão conceitual;
- mova para `04 - Referências` o que for material de consulta futura.

## Princípio operacional
A Inbox não é arquivo definitivo. Ela é uma zona de entrada, não de permanência.
