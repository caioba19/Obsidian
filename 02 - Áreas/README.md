# 02 - Áreas

## Definição
Responsabilidades de longo prazo que exigem a manutenção de um padrão contínuo, sem uma data de término, como Saúde, Finanças ou Desenvolvimento Profissional.

## Propósito
Esta pasta organiza compromissos permanentes da vida e do trabalho. Diferente de projetos, áreas não terminam com uma entrega única: elas exigem manutenção, acompanhamento e evolução constante.

## Regras de Formatação
- Utilize Mapas de Conteúdo (MOCs) para criar índices dinâmicos.
- Estruture cada subárea de modo a facilitar navegação e revisão.
- Agrupe links úteis, recursos, rotinas, checklists e notas relevantes.
- Use os MOCs como painéis de visão geral de cada domínio contínuo.

## O que pertence aqui
- responsabilidades recorrentes
- padrões a manter
- sistemas pessoais
- documentação viva de domínios contínuos

## Critérios de Movimentação
- Se uma Área gerar um objetivo com prazo definido, crie uma nova pasta em `01 - Projetos`.
- Se você deixar de ser responsável por uma Área, mova-a para `05 - Arquivos`.
- Se uma nota desta pasta amadurecer para conhecimento conceitual e reutilizável, sintetize-a também no `03 - Zettelkasten`.

## Princípio operacional
Áreas preservam continuidade. Elas funcionam como centros de manutenção da vida prática, sustentando padrões ao longo do tempo.
