# 03 - Zettelkasten

## Definição
O Motor de Insight. Uma biblioteca permanente focada em pensamento profundo e na síntese de conhecimento, formada por uma rede não hierárquica de notas.

## Propósito
Esta pasta existe para transformar informação em entendimento. Aqui vivem ideias destiladas, conectáveis e reutilizáveis, escritas para apoiar raciocínio de longo prazo e geração de novos insights.

## Regras de Formatação
- Escreva exclusivamente Notas Atômicas.
- Cada nota deve conter apenas uma ideia central.
- Redija sempre com suas próprias palavras para assegurar entendimento real.
- Conecte cada nova nota à teia já existente usando links bidirecionais extensivos.
- Prefira clareza conceitual a acúmulo de material bruto.

## Características de uma boa nota atômica
- uma ideia por nota
- título claro
- texto autoral
- contexto suficiente para releitura futura
- links para notas relacionadas
- valor reutilizável fora do contexto original de captura

## Relação com PARA
As notas do Zettelkasten não devem ser subordinadas às estruturas acionáveis do PARA. Elas servem projetos e áreas por referência, mas permanecem independentes.

## Critérios de Movimentação
As notas aqui são permanentes e atemporais.
- Nunca devem ser movidas para as pastas do PARA.
- Quando for trabalhar em um projeto, apenas referencie estas notas por link.
- Quando uma nota de referência ou de projeto revelar um insight duradouro, crie uma nova nota atômica aqui.

## Princípio operacional
O objetivo não é armazenar tudo, mas construir uma rede viva de ideias que converse consigo mesma ao longo do tempo.
